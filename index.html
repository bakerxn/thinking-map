    📚 For Class Discussion

    • Small Group Activity: In groups of 3-4, trace the path you actually took in your last research paper. Where did it diverge from this map?
    • Debate Prompt: Should "Drafting" be blue or purple? Make your case using examples from your own writing process.
    • Thought Experiment: If you could only use AI for THREE nodes, which would you choose and why?
    • Metacognitive Journal: Track one week of academic work. At which nodes did you actually pause and think vs. operate on autopilot?
    • Redesign Challenge: Create your own version of this map for a different domain (e.g., scientific research, creative writing, data analysis).

    Tools Used

    • Claude (Anthropic) - Conceptual framework development, content generation, code structure
    • Human (Research & Engagement Librarian) - Pedagogical design, course alignment, final editorial decisions

    Process

    1. Human Input: The librarian provided a course description, learning objectives, and initial concept for an interactive map exploring cognitive labor in academic writing.

    2. AI Assistance: Claude generated the node structure, content for each stage of the writing process, and the HTML/CSS/JavaScript implementation. It suggested the constellation metaphor and color-coding system.

    3. Human Refinement: The librarian reviewed all content for accuracy, pedagogical appropriateness, and alignment with course philosophy. Reflection prompts and class discussion activities were collaboratively developed.

    What Was Human-Driven

    • Pedagogical goals and learning outcomes
    • Decision to frame thinking as distributed across tasks (not binary human/AI)
    • Choice to avoid prescriptive "right answers"
    • Integration with specific course themes (philosophy, cognitive science, organizational behavior)
    • Final approval of all content and design choices

    What AI Generated

    • Initial node descriptions and categorizations
    • Technical implementation (HTML/CSS/JavaScript)
    • Visual design suggestions and layout
    • Drafts of reflection questions and discussion prompts

    Why This Transparency Matters

    This page teaches critical thinking about AI collaboration by modeling it. Students see that meaningful work involves human judgment about goals, values, and context—even when AI handles technical execution. The map itself is a product of the process it describes.
